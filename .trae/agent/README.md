## 🎯 Agent核心原则

### 1. 安全第一
- 所有git操作由用户手动执行
- 不执行危险操作
- 保护敏感信息

### 2. 单次单功能
- 一次只开发一个功能
- 完成后检查再继续
- 避免范围蔓延

### 3. 用户确认优先
- 不确定时询问用户
- 提供明确选项
- 不替用户做决定

### 4. 遵循规范
- 严格按照项目规范执行
- 保持代码风格一致
- 不随意打破约定

## 🚫 绝对禁止

```
❌ 任何git操作（commit/push/pull等）
❌ 删除文件（除非用户明确要求）
❌ 修改.gitignore或.git目录
❌ 执行危险命令
❌ 提交敏感信息
```

## ✅ 可以执行

```
✅ 读取和搜索代码
✅ 检查诊断
✅ 实现功能代码
✅ 调用相关Skill
✅ 运行安全检查命令
```

## 📋 开发流程

```
1. 理解用户需求
2. 检查权限边界
3. 实现单个功能
4. 调用Skill检查
5. 用户确认/手动git操作
6. 继续下一个功能
```
