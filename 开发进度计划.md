## 🚀 项目启动指南

### 后端启动

1. 配置数据库：
   ```bash
   cd backend
   cp .env.example .env
   # 编辑 .env 文件配置数据库连接
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 初始化数据库：
   ```bash
   prisma generate
   prisma migrate dev --name init
   ```

4. 启动服务器：
   ```bash
   python manage.py runserver
   ```

### 前端启动

1. 安装依赖：
   ```bash
   cd frontend
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```
